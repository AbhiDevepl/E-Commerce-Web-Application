Project Goal:
Create a basic online store with product catalog, cart, checkout, user authentication, role-based access, product management, order tracking, backend APIs, and MongoDB database integration.

Final Tech Stack:
- Frontend: React + Vite + Tailwind CSS + shadcn/ui
- Routing: React Router DOM
- Backend: Node.js + Express.js + TypeScript
- API Style: REST or ts-rest
- Database: MongoDB
- ORM/ODM: Mongoose
- Authentication: JWT + bcrypt.js
- Validation: Zod
- Environment Config: dotenv
- Package Manager: Bun

Auth APIs:
- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/profile

Product APIs:
- GET /api/products
- GET /api/products/:id
- POST /api/products
- PUT /api/products/:id
- DELETE /api/products/:id

Order APIs:
- POST /api/orders
- GET /api/orders/my-orders
- GET /api/orders/:id
- GET /api/orders
- PUT /api/orders/:id/status

Rules:
- Product create/update/delete APIs must be admin-only.
- Get all orders and update order status must be admin-only.
- My orders should return only logged-in user orders.
- Use proper HTTP status codes.
- Use centralized error handling.
- Use Zod validation for request bodies.
- Use environment variables for MongoDB URI, JWT secret, port, and client URL.
